# 🎓 UniHire – Student Job Portal

UniHire is a simple and user-friendly front-end web application designed to help students explore and apply for job opportunities easily.

---

## 🚀 Features

- 🔍 Search jobs by title or keyword
- 🎯 Filter jobs by category/location
- 💼 Job cards with details (title, company, location)
- 🖱️ Apply button interaction
- 👤 Student profile page
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---
